Project title : Cloud-Native Analytics for Customer Retention and Price Optimization

Northwind Intelligence Platform | Azure Databricks, SQL, Advanced Analytics | GitHub

- Built cloud-native analytics platform on Azure Databricks processing 831 orders and 
  92 customers, implementing 15 advanced SQL queries including window functions 
  (ROW_NUMBER, LAG, PARTITION BY), CTEs, and complex multi-table joins demonstrating 
  expertise in cloud data platforms
- Developed comprehensive business analytics including customer lifetime value analysis, 
  cohort retention tracking, year-over-year growth metrics, and market basket analysis 
  using advanced SQL techniques on Delta Lake storage format
- Created unified analytics view combining 7 tables with 30+ metrics for real-time 
  business intelligence, enabling revenue analysis ($1.26M total), customer segmentation, 
  and product performance monitoring across 8 categories

Tech Stack: Azure Databricks, SQL, Delta Lake, Window Functions, CTEs
